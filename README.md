# 🚗 Car Sales Analysis Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Car Sales Analysis Dashboard** created using **Power BI**. It provides valuable insights into car sales data by analyzing trends, customer behavior, and business performance.

The dashboard helps users make **data-driven decisions** by visualizing key metrics in an easy-to-understand format.

---

## 🎯 Objectives

* Analyze overall car sales performance
* Identify top-selling car models and brands
* Understand regional sales distribution
* Track monthly and yearly sales trends
* Evaluate revenue and profit performance

---

## 📊 Dashboard Features

* 📈 **Sales Trend Analysis** (Monthly & Yearly)
* 🏆 **Top-Selling Cars & Brands**
* 🌍 **Region-wise Sales Distribution**
* 💰 **Revenue & Profit Insights**
* 🎯 **Interactive Filters & Slicers**

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculations and measures
* **Excel / CSV** – Dataset

---

## 📁 Dataset Description

The dataset includes the following fields:

* Car Model
* Brand
* Price
* Quantity Sold
* Sales Date
* Region


---

## 📈 Key Insights

* Sales peak observed during festive seasons
* Certain brands dominate specific regions
* High-priced cars contribute significantly to revenue
* Seasonal trends impact customer purchasing behavior
